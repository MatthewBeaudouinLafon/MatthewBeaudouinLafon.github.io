Virtual Enigma Machine

The Enigma is a WWII electromechanical encryption machine used by the German military.

For my International Baccalaureate's Extended Essay, I chose to explore the mathematical aspect of the machine.

Implementation

During my gap year at NYU, I took a class that heavily used MatLab. To get ahead and more familiar with the language, I decided to program the Enigma. Challenges. MatLab turned out to be a good choice for a language. The algorithm is not very taxing and dealing with permutations is ridiculously easy in MatLab.

Once I had written the code, I thought it would be interesting to make a GUI to use it. I ended up using GUIDE, MatLab's built in GUI maker. Pic.
